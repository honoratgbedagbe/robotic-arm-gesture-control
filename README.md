# 4-Axis Robotic Arm with Gesture Control

**IFRI Robotics Group — 2026**

## Objective

Build a 4-DOF robotic arm controlled in real time by a sensor-equipped glove,
using Bluetooth communication and an STM32F407 microcontroller.

## Team and Roles

- **Nelson / Josué** : 3D design + glove electronics
- **Laurent** : Arm hardware (wiring, power, LCD, buttons)
- **Ronice** : Arduino Nano firmware (glove sensors + Bluetooth)
- **Honorat** : STM32 firmware + project coordination

## Repository Structure

- `Core/` — STM32CubeIDE project files and application code
- `docs/` — Technical documentation
- `hardware/` — Schematics and BoM
